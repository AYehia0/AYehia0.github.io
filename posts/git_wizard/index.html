Git for hackers

Once a wise man said:
“There are two types of hackers in this world: those who know git, and those who still don’t git it”

When I refer to hackers, I mean those who love to go be behind the things to discover how things work and break them (somehow). Originally, hacker simply meant advanced computer technology enthusiast (both hardware and software) and adherent of programming subculture who used their skill to achieve a goal or overcome an obstacle.

If you don’t use git, you should start learning it now!

Disclaimer

This isn’t a tutorial on how to use git or it’s going to make to the best git user but will inspire you to make your git experience better.

If you don’t know what’s git or version control, you should read this book : Pro Git from git’s offical website.

I also assume you have git installed on your machine, and you have some basic experience with Linux Shell.
